# Bank Account Simulator

A console-based banking application simulating core account operations through an interactive menu.

## Features
- Check balance
- Deposit and withdraw funds
- Tracks transaction history

## How to run
1. Clone the repo: `git clone https://github.com/samihasoyshee/bank-account-simulator.git`
2. Run: `python main.py`
